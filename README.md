# cr-nginx

## Quick Start

### Init Development Environment

```shell
pre-commit install
```

### List All Available Tasks

```shell
task -a
```

### Build Container

```shell
containers/latest/build.sh ghcr.io/lwmacct/260629-cr-nginx:latest
```
